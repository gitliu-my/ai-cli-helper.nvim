## [0.1.0] - 2026-02-04
### Added
- Copy path with lines from current buffer (normal/visual).
- Copy file path without line numbers.
- Send path or file path to cursor-agent terminal.
- Focus cursor-agent terminal.
- Default keymaps under `<leader>a`.
